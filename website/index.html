What It Solves

官网先把你最关心的三个问题讲清楚。

有没有公网地址

现在没有内置公网服务。`deployd` 是你自部署的服务端，公网地址由你的服务器决定。

CLI Manual

CLI 说明书，不需要跳文档。

1. 启动 deployd

deployd serve \
  --listen :7319 \
  --data-dir ./.deployctl-data \
  --admin-secret dev-secret

2. 创建 token

deployd admin create-token \
  --data-dir ./.deployctl-data \
  --admin-secret dev-secret \
  --name ci-bot \
  --scope read-only

3. 写入本地配置

deployctl auth token set <token>

4. 检查健康度

deployctl --json doctor

5. 查看当前身份

deployctl --json auth whoami

Token 来源优先级

1. --token
2. DEPLOYCTL_TOKEN
3. ~/.deployctl/config.toml
Agent Handoff

把这些内容直接交给服务器 agent。

推荐机器态 JSON

{
  "project": "deployctl",
  "service": "deployd",
  "public_service_builtin": false,
  "release_url": "https://github.com/ckken/deployctl/releases/tag/v0.1.0",
  "binary_assets": {
    "linux_amd64": "https://github.com/ckken/deployctl/releases/download/v0.1.0/deployctl_linux_amd64.tar.gz",
    "linux_arm64": "https://github.com/ckken/deployctl/releases/download/v0.1.0/deployctl_linux_arm64.tar.gz"
  },
  "required_outputs": [
    "public_base_url",
    "doctor_json",
    "whoami_json",
    "token_scope"
  ],
  "recommended_commands": [
    "deployctl --json doctor",
    "deployctl --json auth whoami"
  ]
}
