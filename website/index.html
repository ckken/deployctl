deployctl · admin homepage

用一个首页，把 admin 管理、用户 token 和 agent 接管链接组织起来。

这里不是文档页，而是 deployctl 的首页管理台。管理员输入 deployd 服务域名和 `adminKey` 后，可以直接创建 token、生成 share link，并把链接交给 agent 自助接管。

当前产品形态
  • 前端：GitHub Pages 首页管理台
  • 后端：deployd API
  • 认证：adminKey + token only
  • 接管：share link -> claim token
