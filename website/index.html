deployctl · admin homepage

把 token 管理和 agent 分享，压缩成一个 dashboard。

这里不是文档页，而是 deployctl 的管理首页。连上 deployd 之后，先创建或选择一个已有 token，再用最少字段直接生成给 agent 的分享链接。

当前产品形态
  • 前端：首页由 deployd 同源托管
  • 后端：deployd API
  • 认证：adminKey + token only
  • 接管：短链接 /s/<share_code> -> claim token
